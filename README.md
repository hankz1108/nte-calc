# 異環素材計算器
用於計算異環素材用的工具  
由於開發原因是找不到符合需求的工具，所以如果未來有出現更符合我需求的工具我將不會繼續維護。

## 網站連結
https://hankz1108.github.io/nte-calc

## 離線使用
本系統原先是為了可以在自己電腦上使用  
所以寫法上允許直接在local端的電腦上直接用瀏覽器打開`docs/index.html`就可以使用

## 功能
### 已有功能
- 角色升級素材計算
- 弧盤升級素材計算
- 角色與弧盤合併總計（共用素材：甲硬幣、殘渣材料自動加總）
